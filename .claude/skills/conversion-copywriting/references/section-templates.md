# Section Copy Templates

## How to Use This Reference

These templates provide proven structures for common page sections. Use them as frameworks, then customize with your ICP's specific pain points, language, and brand voice.

---

## Hero Section

### Template 1: Value Proposition Hero

```
HEADLINE: [Primary benefit in their language]

SUBHEADLINE: [Expand with who it's for + how it works]
             (1-2 sentences max)

SUPPORTING COPY (optional):
[One paragraph that elaborates on the value. 
Address the "what" and "for whom" if not in headline.]

PROOF ELEMENT:
[Quick trust: "Trusted by 10,000+" OR star rating OR 
one-sentence testimonial]

PRIMARY CTA: [Main action button]
SECONDARY CTA: [For not-ready: "See how it works"]
```

**Example:**
```
HEADLINE: Ship code 3x faster without the DevOps headaches.

SUBHEADLINE: The deployment platform built for engineering teams 
who'd rather build features than fight infrastructure.

PRIMARY CTA: "Start Free Trial"
SECONDARY CTA: "Watch 2-Min Demo"

PROOF: ★★★★★ "Finally, deployment that just works." 
       — 4.9 from 2,800+ reviews
```

## Problem Section

### Template 1: Pain Point List

```
HEADLINE: [Acknowledge the collective pain]

INTRO PARAGRAPH:
[1-2 sentences validating their frustration]

PAIN POINTS (3-5):
• [Pain point in their language]
• [Pain point in their language]
• [Pain point in their language]

TRANSITION:
[1 sentence hinting at solution]
```

**Example:**
```
HEADLINE: Sound familiar?

INTRO: If you're like most agency owners, 
       you've probably experienced:

• Scope creep eating into your margins
• Clients asking "what's the status?" constantly
• Team members duplicating work because nobody knows who's doing what
• Projects going over budget with no warning

TRANSITION: What if there was a better way?
```

### Template 2: Scenario Description

```
HEADLINE: [Relatable scenario question or statement]

BODY:
[2-3 paragraphs painting the picture of their 
current painful reality in vivid detail]

[Use second person "you" to make it personal]
[Include specific details that make them nod]

TRANSITION:
[Segue into solution]
```

**Example:**
```
HEADLINE: Monday morning. The inbox is already on fire.

BODY:
You wanted to start the week with strategy. Instead, you're 
putting out fires. Three clients emailed over the weekend. 
Your designer is waiting on feedback you haven't had time to give. 
And somewhere in Slack, there's a thread about a deadline 
you're pretty sure was yesterday.

You didn't start an agency to spend your days in reactive mode. 
But somehow, that's where you are.

TRANSITION: Here's the thing: It doesn't have to be this way.
```

## Objection Handling Section

### Template 1: FAQ Format

```
HEADLINE: Questions? We've got answers.

Q: [Objection phrased as natural question]
A: [Direct, honest answer with proof if available]

Q: [Objection phrased as question]
A: [Direct answer]

Q: [Objection phrased as question]
A: [Direct answer]

FALLBACK:
"Still have questions? [Contact link]"
```

### Template 2: Direct Address

```
HEADLINE: [Address the elephant in the room]

"You might be wondering: [exact objection]"

[2-3 sentences addressing it directly and honestly]
[Include proof point if available]

"Still skeptical? [Risk reversal or proof offer]"
```

**Example:**
```
HEADLINE: "But will it work for MY industry?"

You might be wondering if this is really built for [industry]. 
Fair question—we get it a lot.

Here's the truth: We have 500+ [industry] teams using the 
platform right now. Companies like [example], [example], 
and [example] rely on us daily.

Still skeptical? Book a demo and we'll show you exactly 
how teams in your industry use it.

CTA: "See [Industry] Examples"
```

## Final CTA Section

### Template 1: Simple Closer

```
HEADLINE: [Restate primary benefit]

SUPPORTING:
[1-2 sentences—quick reminder of value 
and/or address lingering doubt]

CTA: [Strong action button]

RISK REVERSAL:
[Guarantee, free trial, cancel anytime]
```

**Example:**
```
HEADLINE: Ready to ship faster?

SUPPORTING: Join 10,000+ engineering teams who've eliminated 
            deployment stress. Try it free for 14 days.

CTA: "Start Free Trial"

RISK REVERSAL: No credit card required. Cancel anytime.
```

### Template 2: Urgency Closer (When Authentic)

```
HEADLINE: [Time-sensitive offer]

SUPPORTING:
[Explain the urgency/scarcity]
[Why they should act now]

CTA: [Urgent action]

DEADLINE:
[Specific deadline or countdown]
```
