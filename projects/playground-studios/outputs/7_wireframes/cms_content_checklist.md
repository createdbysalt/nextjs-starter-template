## Image Optimization Guidelines

Before uploading:
1. **Resize** to web dimensions (max 2000px on longest edge for most)
2. **Compress** using tools like TinyPNG or Squoosh
3. **Name descriptively** (e.g., `celsius-campaign-01.jpg`)
4. **Use JPG** for photos, **PNG** for graphics with transparency

Target file sizes:
- Hero images: Under 500KB
- Gallery images: Under 300KB each
- Logos: Under 50KB (SVGs are usually tiny)
